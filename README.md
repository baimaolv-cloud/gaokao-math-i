# 新高考Ⅰ卷·数学备考工具

面向浙江（及共用新高考Ⅰ卷省份）高三学生的**纯前端、零后端、可离线**数学备考工具。
所有内容按《普通高中数学课程标准》五大主题组织，聚焦"新高考Ⅰ卷"考点、真题与易错点。

## 包含的版本

| 文件 / 目录 | 形态 | 用法 |
|---|---|---|
| `新高考Ⅰ卷数学考点知识图谱.html` | 知识图谱（可交互） | 双击即开，按五大主题看考点关联 |
| `新高考Ⅰ卷数学考点精炼与真题演练.html` | 主程序（在线版，MathJax 走 CDN） | 双击即开；联网时公式渲染最佳 |
| `新高考Ⅰ卷数学考点精炼与真题演练(离线版).html` | **单文件离线版** | 一个文件 = 完整产品，MathJax 已内联，零外链，双击即用 |
| `gaokao-math-local/` | **文件夹版（PWA）** | 起本地服务或部署到 https →「添加到主屏幕」→ 像 App 全屏 + 离线 |

> 两个离线版功能一致：21 个核心考点 · 细分知识 · 高考真题演练（2024/2025 新高考Ⅰ卷）· 易错警示与混淆辨析（口诀妙法）· 参数动态演示（可拖动滑块）· 可打印参数探究题单。区别只在**交付形态与安装体验**。

## 各版本怎么选

- **发给客户 / 家长，随手打开** → 用单文件离线版（最省事，免维护目录）
- **机房 / 内网批量部署、要"像 App"** → 文件夹版 + 本地服务器（`python3 -m http.server 8000`）
- **上公司网站对外分发、要裂变留存** → 文件夹版部署到 https 站点（PWA 主屏安装 + 二次离线）

## 本地运行（文件夹版 PWA）

```bash
cd gaokao-math-local
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

⚠️ PWA 的「添加到主屏幕 / 离线」只在 **https 或 localhost** 源生效，双击 `file://` 打开不会触发。

## 内容说明

- 真题来源逐年、题面以官方公布为准，本工具仅供参考，不作提分承诺。
- 公式渲染使用 [MathJax](https://www.mathjax.org/)（离线版已本地化，无外部依赖）。
- 单文件离线版与文件夹版均为纯前端：无账号、无后端、不上传任何用户数据。
