# 新高考Ⅰ卷 · 数学备考工具（离线版）

本包为**纯前端、零外链、可离线**的高考数学备考工具，适合直接拷给客户在本地电脑 / 机房 / 内网使用。

## 目录结构
- `index.html` —— 主程序（双击即可在浏览器打开，完全离线运行）
- `vendor/mathjax-tex-svg.js` —— 公式渲染引擎（已本地化，无需联网）
- `manifest.json` —— PWA 清单（名称、图标、主屏安装配置）
- `sw.js` —— Service Worker（离线缓存，断网也能打开）
- `icon.svg` / `icon-maskable.svg` —— 主屏图标
- `README.md` —— 本说明

> 另提供**单文件版** `新高考Ⅰ卷数学考点精炼与真题演练(离线版).html`（MathJax 已内联），
> 一个文件即可发给客户，免维护目录结构，按需取用。**注意：单文件版无法做成 PWA**（Service Worker 需要被服务的源，不能用 `file://` 双击触发）。

### ✅ 已升级为 PWA
- 支持**「添加到主屏幕」**：在手机/电脑浏览器菜单里添加后，像原生 App 一样全屏打开，无地址栏。
- 支持**离线**：首次联网打开后，缓存应用外壳，之后断网也能正常使用。

### ✅ 已加入首页导航
- 顶部 **sticky 导航栏**：首页 / 全部考点 / 函数与导数 / 几何与代数 / 概率与统计 / 易错辨析 / 动态演示，点击即筛选并平滑滚动到对应模块。
- 首页 **hero 模块卡片**：考点精炼、真题演练、易错辨析、参数动态演示、参数探究题单（一键导出），点卡片直达。
- 单文件离线版与文件夹版均已同步此导航；单文件版双击即用，文件夹版走 http 源即含 PWA 能力。

## 使用方法
**方式一（最简单）**：直接双击 `index.html`，用任意现代浏览器（Chrome / Edge / 360 / Safari）打开即可。
所有功能——考点细分、真题演练、易错辨析、动态演示、参数探究题单导出——均离线可用。

**方式二（推荐，规避个别浏览器对 `file://` 的限制）**：在本文件夹下启动一个本地服务器：
```bash
python3 -m http.server 8000
```
然后浏览器访问 `http://localhost:8000` 即可。

## 已做的离线化处理
- 公式引擎 MathJax 已从公共 CDN 改为**本地 `vendor` 引用**（单文件版直接内联），整包**零外链**。
- 已**关闭 MathJax 无障碍语音引擎（a11y）**，避免离线环境下尝试联网加载语音规则导致控制台报错。

## 进阶：PWA 详细使用方式

### ⚠️ 关键前提
PWA 的「添加到主屏幕」和「离线」**必须在一个被服务的源（https 或 localhost）上运行**，
**不能靠双击 `index.html`（`file://`）触发**——这是 Service Worker 的安全限制，不是 bug。
所以 PWA 用法一律走「方式二：本地服务器」或部署到网站。

### 场景 A：本地 / 机房 / 内网（无公网域名）
1. 在本文件夹下启动本地服务器：
   ```bash
   python3 -m http.server 8000
   ```
2. 浏览器访问 `http://localhost:8000`（同局域网其他设备用 `http://<本机IP>:8000`）。
3. **首次打开需联网一次**（让 Service Worker 缓存应用外壳）。
4. 之后断网也能正常打开；手机端在浏览器菜单选择「添加到主屏幕」即可生成桌面图标。

### 场景 B：部署到公司网站（推荐对外分发）
1. 把整个 `gaokao-math-local/` 文件夹上传到公司站点子目录（如 `公司域名/gaokao-math/`）。
2. 必须走 **https**（公司站通常已具备）。Service Worker 只在 https 或 localhost 生效。
3. 访问该地址，浏览器会自动提示「安装 / 添加到主屏幕」。
4. 学生扫码或点链接打开 → 添加主屏 → 像 App 一样离线使用，留存与触达都优于单文件。

### 学生视角的「使用方式」（最简说明，可直接转发）
> 1. 用手机/电脑浏览器打开老师给的链接（或局域网地址）。
> 2. 联网打开一次后，点浏览器菜单「添加到主屏幕 / 安装」。
> 3. 桌面上会出现「高考数学精炼」图标，点开就是全屏 App，没网也能用。

### 验证 PWA 是否生效（开发者自查）
- Chrome 桌面端：打开 `http://localhost:8000` → F12 → Application 面板 →
  Manifest（应读到名称/图标）、Service Workers（应显示 `sw.js` 已激活）、Cache Storage（应有 `gaokao-math-v1`）。
- 手机端：地址栏右侧出现「安装」图标，或菜单中有「添加到主屏幕」。
- 离线测试：F12 → Network 勾选 Offline → 刷新页面，内容仍正常加载即成功。

## 如需微调
- 改应用名 / 图标：编辑 `manifest.json` 与 `icon.svg`（也可替换为 192/512 PNG 以兼容更多设备）。
- 改缓存版本：升级内容后把 `sw.js` 里的 `CACHE = 'gaokao-math-v1'` 改为 `v2`，旧缓存会自动清理。
